# Monitor Bot

This bot will help monitor data from websites.

## Installation

Use the package manager [pip](https://pip.pypa.io/en/stable/) to install requirments.

```bash
pip install -r requirments.txt
```

## Usage

To run the program, please run this line.

```bash
python main.py 
```
